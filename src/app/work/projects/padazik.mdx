## Aperçu

**PadaZik** est un lecteur de musique simple développé en Java avec le framework **LibGDX**. Ce projet a été réalisé dans le but principal de découvrir et d'apprendre à utiliser LibGDX pour le développement d'applications graphiques et de jeux.

## Fonctionnalités

- **Lecture Audio** : Support de base pour la lecture de fichiers musicaux.
- **Interface Vinyle** : Une interface utilisateur stylisée autour du concept de disques vinyles (classe `Vinyle.java`), offrant une esthétique rétro.
- **Gestion d'Albums** : Organisation de la musique par albums.

## Aspects Techniques

Le projet est construit autour de l'architecture standard de LibGDX :

- **Java & LibGDX** : Le cœur du projet.
- **Gestion des Assets** : Utilisation d'un `AssetsManager` pour charger efficacement les ressources graphiques et sonores.
- **Écrans** : Utilisation du système de `Screen` de LibGDX, avec un `MenuScreen` principal.
