## Apercu

**Parcoursup** est une application web qui facilite l'exploration des formations via l'API Open Data de l'enseignement superieur.

L'objectif est de filtrer rapidement les resultats selon plusieurs criteres (nom, filiere, ville, departement, moyenne estimee et selectivite), puis de visualiser les etablissements sur une carte.

## Fonctionnalites principales

- **Recherche multi-criteres** sur les donnees Parcoursup.
- **Pagination** des resultats pour une navigation plus lisible.
- **Carte interactive** (OpenStreetMap + Leaflet) avec marqueurs geolocalises.
- **Calcul de moyenne estimee** a partir des donnees d'admission.
- **Filtrage de selectivite** avec tranches de taux d'acces.

## Stack technique

- **Riot.js** pour l'interface et les composants.
- **JavaScript** pour la logique metier et les appels API.
- **Open Data API ESR** pour la source de donnees.
- **Leaflet + OpenStreetMap** pour la visualisation geographique.
- **CSS** pour le design responsive.
