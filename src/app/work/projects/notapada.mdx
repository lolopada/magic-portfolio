## Aperçu

**NotaPada** est une application de bureau conçue pour offrir une expérience de prise de notes fluide, initialement développée pour apprendre à maîtriser **Electron**.

## Fonctionnalités Clés

- **Gestion de Fichiers Avancée** : 
  - Création, renommage et suppression de fichiers et dossiers directement depuis l'interface.
  - Système de navigation intuitif dans l'arborescence.
- **Éditeur Intelligent** :
  - **Sauvegarde Automatique** : Ne perdez jamais votre travail. Chaque frappe est sécurisée.
- **Stockage Local** : Vos données vous appartiennent, stockées localement dans un dossier `userdata` configurable.

## Technologies Utilisées

- **Electron** : Pour le packaging de l'application cross-platform.
- **Node.js** : Gestion du système de fichiers (fs) et backend local.
- **HTML5 / CSS3** : Interface utilisateur responsive et stylisée.
- **JavaScript** : Logique de l'application et manipulation du DOM.

## Installation

L'application est conçue pour être simple à installer et à configurer pour les développeurs :

```bash
git clone https://github.com/lolopada/NotaPada.git
cd NotaPada
npm install
npm start
```
